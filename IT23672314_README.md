# IT3040 ITPM - Assignment 1
**Registration Number:** IT23672314

This repository contains the Playwright automation script to test the chat-translator transliteration application.
The objective is to identify fifty test cases where the system fails to correctly convert chat-style Singlish into Sinhala, and automate their execution.

## Prerequisites
- Python 3.11 or 3.12
- pip (Python package installer)

## Installation

1. Clone this repository (or download and extract the ZIP file).
2. Open a terminal or Command Prompt and navigate to the project directory.
3. Install the required dependencies by running:
   ```sh
   pip install playwright openpyxl
   playwright install
   ```

## Running the Tests

To run the automated Playwright tests, execute the following command in the project directory:

```sh
python test_automation.py --excel "Assignment 1 - Test cases.xlsx" --url "https://www.pixelssuite.com/chat-translator" --headless
```

Note: If you want to watch the browser actions visually, you can remove the `--headless` flag.

## Project Structure
- `test_automation.py`: The Playwright script provided for automated testing.
- `Assignment 1 - Test cases.xlsx`: The Excel file containing the 50 identified negative test cases and their execution results.
